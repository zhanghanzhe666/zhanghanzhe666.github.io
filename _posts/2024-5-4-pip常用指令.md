## 前言
pip是python的原版包管理器，可以让python安装上第三方模块，增加python的功能，如numpy,pandas。
## 正文
通常，你可以在cmd中输入`pip`来查看pip的信息。
以下为pip对以下命令的解释
- `install`：安装包。  
- `download`：下载包。  
- `uninstall`：卸载包。  
- `freeze`：以requirements格式输出已安装的包。  
- `inspect`：检查Python环境。  
- `list`：列出已安装的包。  
- `show`：显示已安装包的信息。  
- `check`：验证已安装的包是否具有兼容的依赖关系。  
- `config`：管理本地和全局配置。  
- `search`：在PyPI中搜索包。  
- `cache`：检查和管理pip的wheel缓存。  
- `index`：检查可从软件包索引获取的信息。  
- `wheel`：从您的需求构建轮。  
- `hash`：计算包存档的哈希值。  
- `completion`：用于命令完成的辅助命令。  
- `debug`：显示用于调试的有用信息。  
- `help`：显示命令的帮助信息。  
接下来是一些常用指令使用方法。
#### install  
```bash
pip install 包名
#同时，install 命令可以加一些参数
pip install -r 名称.txt #安装'名称.txt'所有指定的包
pip install -i 镜像源 #用指定镜像安装包
pip install 包名 == 版本号 #安装指定版本的包
```
#### uninstall  
```bash
pip uninstall 包名
```
#### show  
```bash
pip show 包名
```
#### list  
```
pip list #以列表列出安装的包
```
#### config

使用pip config命令可以轻松地配置pip，以下是使用方法：  

1. 查看当前配置
要查看当前pip的配置，可以使用以下命令：
```
pip config list
```
这会列出当前配置的所有选项及其对应的值。  
3. 设置配置选项  
要设置配置选项，可以使用以下命令：  
```
pip config set <key> <value>
```
其中<key>是要设置的配置选项，<value>是对应的值。  
例如，要设置默认的包安装源为清华大学的镜像：
```
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple`
```
4. 获取特定配置选项的值  
要获取特定配置选项的值，可以使用以下命令：
```  
pip config get <key>
```
其中<key>是要获取值的配置选项。  
例如，要获取默认的包安装源：  
```
pip config get global.index-url
```
6. 重置配置选项
要重置配置选项，可以使用以下命令：
``` 
pip config unset <key>
```
其中<key>是要重置的配置选项。  

例如，要重置默认的包安装源：  
```
pip config unset global.index-url
```
5. 配置文件  
pip的配置信息保存在用户的主目录下的.pip文件夹中的pip.conf或pip.ini文件中。可以直接编辑这些文件来修改配置，也可以使用pip config命令来修改。
####  freeze
```
pip freeze 文件名.txt#将当前python环境中的所有库打包成一个txt文件
```
#### `pip`升级  
要升级pip，可以使用以下方法：  
1. 使用pip命令升级  
可以通过以下命令使用pip自身来升级：
```
pip install --upgrade pip
```
这会使用pip来下载并安装最新版本的pip。  
3. 使用python -m pip命令升级  
另一种方法是使用python -m pip来执行pip的升级：
```
python -m pip install --upgrade pip
```
这样可以确保使用的是与当前Python解释器关联的pip版本。   
3. 在Windows上升级pip  
如果你在Windows上使用pip，你也可以通过以下命令来升级：
``` 
python -m pip install --upgrade pip
```
这会执行相同的升级操作，确保pip被更新到最新版本。  
5. 验证升级  
升级完成后，可以使用以下命令验证pip的版本是否已经更新：
```
pip --version
```
这会显示安装的pip版本信息。   
以上就是pip的基本操作。
